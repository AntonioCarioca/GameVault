The Game Gear is an 8-bit fourth generation handheld game console released by Sega. The Game Gear primarily competed with Nintendo's Game Boy, the Atari Lynx, and NEC's TurboExpress. It shares much of its hardware with the Master System, and can play Master System games through the use of an adapter.
