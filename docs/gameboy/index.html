Game Boy is a handheld game console developed by Nintendo, following the success of the Game & Watch single-game handhelds, Nintendo developed the Game Boy to be more like a portable console, with interchangeable cartridges. The concept proved highly successful and the Game Boy became a cultural icon of the 1990s.
