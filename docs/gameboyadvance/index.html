The Game Boy Advance is a 32-bit handheld game console developed, manufactured, and marketed by Nintendo as the successor to the Game Boy Color. The GBA is part of the sixth generation of video game consoles. The original model was followed in 2003 by the Game Boy Advance SP, a redesigned model with a frontlit screen and clamshell form factor.
